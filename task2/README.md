# Task 2 – Feature Engineering & Model Comparison

This task focuses on improving a machine learning regression pipeline using
feature scaling, multiple models, and performance comparison.

## Objectives
- Apply feature scaling
- Train multiple regression models
- Compare models using RMSE and R²
- Select the best-performing model

## Dataset
California Housing Dataset

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## Evaluation Metrics
- RMSE
- R² Score
