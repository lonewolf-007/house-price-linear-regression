# House Price Prediction using Linear Regression

This repository contains a machine learning project that predicts median house
prices using the California Housing dataset.

## Objective
To build and evaluate a Linear Regression model using a complete ML workflow
including data exploration, preprocessing, training, and evaluation.

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Structure
- task1_ml_linear_regression.ipynb : Main notebook with analysis and model
- report/ : PDF report summarizing results
- linear_regression_model.pkl : Saved trained model (optional)

- Exploratory Data Analysis (EDA)
- Linear Regression baseline
